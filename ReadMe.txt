Ws.conf is the configuration file of the server where the basic configuration settings of the server is saved.
The Server doesnot run without this ws.conf file.
The types of files that the server can handle is given by this ws.conf file.
The server is a multithreaded http server employing the GET and POST method


The GET method gets the file from the server, when the client makes a request from the browser or from other HTTP Clients.
The POST method posts the information from the Client on the HTTP Server.

Client can be a browser or an HTTP Client which displays the web pages, images and links on the client after getting it from the server.
